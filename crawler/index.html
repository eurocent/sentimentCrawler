Sentimentcrawler

sentimentCrawler is a file or http crawler which will build subject, predicate, object triple representations for use in Sentiment Analysis tasks.

Welcome to sentimentCrawler

Sentiment Crawler is a software component generated around the Eurosentiment Platform. In particular Sentiment Analysis is described as a new discipline whose aim is the computational treatment of opinion, sentiment and subjectivity within texts.

The main concept of the Eurosentiment project is to provide a shared language resource pool for fostering sentiment analysis. Sentiment Crawler acts as one of many tools within the Eurosentiment Toolkit for achieving this goal.

SO... What is sentimentCrawler???

It is a Java program and command line tool which can be used to fetch Webpages over HTTP or file protocols and provide extraction capabilities, the output of which can be used for Sentiment Analysis tasks.

The workflow is as follows:

  1. fetch am (HTML) document
  2. extract subject, predicate, object (triples) relationships from implicit content and structure
  3. define an output representation/serialization format for persisting the generated triples
  4. write the triples to an output directory for use within Sentiment Analysis tasks

Current Features

The tool is extremely lightweight but there are some nice features

  1. The output path for generated triples is fully configurable (right now this needs to be locally on file system)
  2. The output format is configurable and can be one of turtle, ntriples, rdfxml, nquads, trix or json
  3. The extraction of implicit structure is handled by Apache Any23, an actively maintained top level projects at The Apache Software Foundation

How can I use sentimentCrawler?

Pretty easily... we did say it was lightweight after all didn't we!:

$ git clone https://github.com/eurocent/sentimentCrawler.git
$ cd crawler
$ mvn install
$ cd target
$ java -jar sentiment-crawler-0.0.1-SNAPSHOT-jar-with-dependencies.jar -url http://reviews.bestbuy.com/3545a/6141277/zagg-invisibleshield-for-samsung-galaxy-gear-2-neo-fit-watch-reviews/reviews.htm -outputDir /usr/local/sentimentCrawler/crawler/target -outputFormat turtle

In the example above, the output will be written to a the file located at /usr/local/sentimentCrawler/crawler/target/sentiment.txt
